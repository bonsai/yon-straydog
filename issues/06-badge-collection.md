# バッジ（玉）コレクションシステム

## 概要
各スポットをクリアすると「玉＝バッジ」を獲得できるコレクションシステムを実装する。

## タスク
- [ ] バッジのデータモデルを定義する
- [ ] スポットクリア時にバッジを付与する
- [ ] バッジ獲得状況を localStorage に保存する
- [ ] バッジ一覧表示画面を作成する
- [ ] 3つのバッジを集めると最終スポットが解放される仕組み

## 関連ファイル
- `src/game/spots.ts` - バッジデータ
- `src/store.ts` - 保存処理（未確認）
