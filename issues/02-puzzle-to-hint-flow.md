# 写真パズル完了 → ヒント表示 → スポット探索のフロー

## 概要
写真パズルを解いた後、ヒントが表示され、ユーザーが実際にその場所を訪れるまでの流れを実装する。

## タスク
- [ ] パズル完了後、`buildHintSteps()` を使ってヒントテキストを表示する
- [ ] 「さぼうる」「響」「神田橋公園」の3スポットのいずれかに誘導する
- [ ] ヒント表示後に `phase: 'play'` へ遷移する
- [ ] 各スポットの座標（lat/lng）を確認し、MAP画面へ移行する
