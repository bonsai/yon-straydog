# 実機テスト項目

## 概要
iOS Safari / Android Chrome での動作確認。特にカメラ・GPS・画面サイズ。

## テスト項目

### カメラ
- [ ] iOS Safari: `getUserMedia` が動作する（playsinline必須）
- [ ] Android Chrome: カメラ起動・切替
