# GPS位置確認とスポット到達検知

## 概要
ユーザーが実際に目的のスポットに到着したかをGPSで確認する機能を実装する。

## タスク
- [ ] Geolocation API を使って現在地を取得する
- [ ] 各スポットの座標と現在地の距離を計算する
- [ ] 一定距離（例: 30m）以内に入ったら「到着」と判定する
- [ ] 到着したら該当のミニゲーム（ぷよ/サイモン/クイズ）を起動する
- [ ] 位置情報のエラーハンドリング（許可がない場合など）

## 関連ファイル
- `src/game/map.ts` - Leaflet マップ
- `src/game/spots.ts` - スポット座標データ

## 備考
現在 `map.ts` は Leaflet で地図表示のみ。GPS監視ロジックは未実装。
