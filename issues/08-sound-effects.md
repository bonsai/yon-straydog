# サウンドエフェクトとBGM

## 概要
各所に音を追加して没入感を高める。Web Audio API で実装（ファイル不要）。

## タスク
- [ ] 犬の鳴き声（bark）: ミニゲームクリア時に再生
- [ ] 正解/不正解音: クイズ・パズル完了時
- [ ] 足音/環境音: マップ画面でループ
- [ ] タイプライター効果音: アドベンチャーテキスト表示中
- [ ] BGM: Goldberg Variations 冒頭をループ（オプション）
- [ ] 音量設定UI: ツールバーに追加

## 実装方針
- Web Audio API の OscillatorNode / Noise で音を合成（ファイル不要）
- または Base64 エンコードした短い音声を埋め込む
- `AudioContext` はユーザーの初回タップ時に resume する（iOS制約）

## 関連
- PRD: F5.2, F5.3
- `src/style.css` に音関連のクラスは不要（JSのみ）
