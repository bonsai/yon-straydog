# yon-walk ARG 4謎 詳細設計

## 全体フロー

```
参加者
  │
  ├─ 1. 七色スタンプラリー（7駅巡り）
  │     → 7色そろう → 金の玉の場所（神田橋公園）が判明
  │
  ├─ 2. Kraut und Rüben（尾張家）
  │     → Var.30のヒントで正解注文 → レシートに暗号
  │
  ├─ 3. 金の玉（神田橋公園・豊展観守像）
  │     → Aria da Capo再生 → 像が「開く」→ 中から黄金のカプセル
  │
  └─ 4. 黒真珠の場所（東京駅）
        → Var.25を聴きながら写真 → 表(金色駅舎)と裏(工業)の対比
```

### スタンプの形態

- **QRコード方式**（推奨）: 各スタンプ位置にA6サイズのQRカード設置
  - 読み取るとスマホに色付きスタンプ画像が追加
  - スマホのスタンプ台紙アプリ or LINE Bot で管理
- **物理スタンプ方式**（代替）: 各位置に紙のスタンプ台 + インク
  - スタンプ帳はさぼうるで配布（500円以上注文で1部）
  - スタンプ帳には7色の円枠だけ印刷 → 参加者が押して埋める

### スタンプデザイン

各駅のスタンプは **「その駅の色に対応するクリームソーダのグラフィック」+「駅名」**。
- 赤: イチゴの実 + C(千代田) ロゴ
- 橙: オレンジスライス + G(銀座) ロゴ
- 黄: レモン輪切り + JB(中央総武) ロゴ
- 緑: メロン網目 + S(新宿) ロゴ
- 青: ブルーハワイ波紋 + T(東西) ロゴ
- 藍: カルピス水玉 + I(三田) ロゴ
- 紫: 巨峰房 + Z(半蔵門) ロゴ

### 7色達成時の演出

全7色を集めると、スマホ画面（またはスタンプ帳の最終ページ）に**神田橋公園・豊展観守像の地図**が出現。
- QR版: 7個目のQR読取時に地図表示
- 物理版: スタンプ帳末尾に隠しページ → 7色押すと浮かび上がる（コーヒー染め or 型抜き）

### 所要時間

- 歩行のみ: 約60分（1駅あたり平均8-9分）
- スタンプ押印含む: 約75分
- 余裕見積: 90分

### 注意事項

- 神田駅（15分）と大手町駅（15分）が最も遠い → 任意順序でOK（最短ルートは神保町→水道橋→小川町→竹橋→大手町→神田→赤御茶ノ水 または その逆）
- 藍（大手町・三田線）のカルピスは白いドリンク＝包み紙の藍色＝ヒント → 3番目の謎につながる伏線

## 2. Kraut und Rüben（尾張家）— 謎の中身設計

### 前提

- 場所: 尾張家（神田鍛冶町1-6-4）
- 営業時間: 17:00-21:30LO / 土日祝休
- 価格: おでん各種300円〜、ロールキャベツ400円
- 参考: Var.30「Kraut und Rüben haben mich vertrieben」
  （キャベツとカブに追い出された）

### 謎の流れ

**Step 1: ヒント入手**
参加者は事前に（スマホ or ガイドブックで）以下のヒントを得る:
> "Var.30 Kraut und Rüben — キャベツとカブに追い出された男は、
>  神田の路地で小さな鍋に出会う。鍋の中には何がある？"

またはより直接的に:
> "バッハが最後に笑った。Kraut(キャベツ)とRüben(カブ)が
>  逃がしてくれない。2つの品を注文せよ。"

**Step 2: 注文**
尾張家のメニューから以下を注文:
1. **ロールキャベツ**（400円）= Kraut
2. **大根**（300円）= Rübe

※「カブ」ではなく「大根」なのは、尾張家のメニューにカブがないため、
  ドイツ語Rübe（根菜全般）の解釈を広げるトリック

**Step 3: 暗号入手**
レシート（またはお品書きの裏）にスタンプ or 手書きで暗号が押される:
- ロールキャベツの串の本数 or 大根の切り口の数 = 数字ヒント
- レシート合計金額（400+300=700円）が合言葉の一部

**Step 4: 暗号解読**
レシートに記載された数字（例: 700 または 7-0-0）を、
ゴールドベルク変奏曲の変奏番号に対応させる:
- 7 = Var.7（ジーガ=跳躍=神保町へ跳べ）
- 0 = Aria / Aria da Capo（主題へ帰還）
- または 700 = 7 + 0 + 0 = 金の玉の場所の絞り込み

### より拡散的な設計（軽量版）

尾張家に入店せずともクリア可能な方法:
- 店のショーケースに「本日のおでん種: ロールキャベツ・大根」の表示を確認
- スマホでその写真を撮って送信 = クリア
- ただし入店注文すると「金の玉」の追加ヒントが得られる

### 注意

- 営業時間（17時〜、土日祝休）をイベントスケジュールと整合させる
- 19時以降の参加者は尾張家に間に合わない可能性 → 軽量版で代替
- 尾張家の店主がこのARGに協力するかは別途確認

## 3. 金の玉（豊展観守像）— オープン演出デザイン

### 場所

神田橋公園（千代田区神田錦町1-29）
豊展観守像（山下恒雄作・1991年 / 金銅鎚起）

### 前提条件

- 7色スタンプラリーを完了していること
- スマホ（音楽再生可能なデバイス）を持っていること

### 演出の流れ

**Step 1: 場所を特定**
7色スタンプラリー完了 → マップに神田橋公園の位置が表示される
「金の玉は神田橋公園にある。コガネムシが待っている」

**Step 2: 像の前でAria da Capoを再生**
参加者が豊展観守像の前に立ち、ゴールドベルク変奏曲の
Aria da Capo（冒頭のアリアの再現）をスマホで再生する。

**Step 3: 「開く」演出（AR or 物理）**

*案A: AR演出（推奨）*
- スマホカメラを像にかざすと、像の腹部が「開き」、中から金色の光が溢れるAR
- 像の腹部分にARマーカーを設置（スマホで認識可能な小さな模様）
- ARの中で金の玉が浮かび上がり、中から「次の謎（黒真珠）のヒント」が出てくる
- 技術: 8th Wall / WebAR（スマホアプリ不要）

*案B: 物理仕掛け（低コスト）*
- 像の台座に小型の鍵付きボックスを設置（一時的設置）
- ボックスは像の前に貼られたQRコードを読み取ると解錠（スマ連動南京錠）
- 中に次へのヒントカード（黒真珠の場所: 東京駅）

*案C: SNS連動*
- 像の前で「#金の玉 #YONARG」のハッシュタグ付き写真を投稿
- すると公式アカウントからDMで次のヒントが返信される

### 演出のコンセプト

**Goldberg の語源を視覚化:**
- Gold(金) + Berg(山/虫) = コガネムシ
- 豊展観守像は金銅のコガネムシ人間
- Aria da Capo（冒頭帰還）を再生することで「主題（原形）への帰還」を演出的に表現
- 7色のスタンプ（7色クリームソーダ） = 虹 = 光のスペクトル
- 虹の光が集まると「金」（黄金）になる（← 物理的には正しくないが詩的な表現）
- 藍色のカルピス包み紙の水玉 = 天の川の星 = 星が集まると金の玉になる

### 所要時間
- 公園到着〜像発見: 5分
- 演出体験: 3-5分
- 合計: 約10分

## 4. 黒真珠の場所（東京駅）— 写真判定ルール

### 場所

v2（正式）: 東京駅（丸の内口 + 八重洲口側）
v1（参照）: 日暮里駅

### 前提

Var.25（黒真珠 / ト短調 / 最深の悲しみ）を聴きながら撮影。

### 撮影ルール

参加者は**1枚の写真**に以下の両方を含める:

**必須要素:**
1. **東京駅丸の内口（赤レンガ＋ドームの金色駅舎）** — 「表の顔」＝Gold(金)
2. **東京駅の八重洲口側 or 線路敷地（駅舎裏の工事現場・高層ビル群）** — 「裏の顔」＝黒真珠(Black Pearl)

**構図の自由度:**
- 1枚の写真に両方収める（丸の内口の前から裏のビル群を望む OR 裏側から表を望む）
- 駅構内の通路を通して両方見える位置（例: 丸の内地下中央口から八重洲方面を見る）
- または駅舎の外観で「表(金色)」と「裏(コンクリート/工事)」が同時に見える角度

### 判定基準

| 要素 | 必要 | 加点 |
|------|------|------|
| 丸の内口の赤レンガ＋ドーム | 必須 | — |
| 八重洲口側のビル群/工事 | 必須 | — |
| 写真内に金色要素がある | — | +（ゴールドベルクとの親和性） |
| 暗いトーン（夕方/夜） | — | +（黒真珠の哀愁） |
| 線路/ホームが写る | +（鉄道らしさ） | — |
| Var.25が再生中の証拠（イヤホン/画面） | — | + |

### 提出・確認方法

**スマホ方式（推奨）:**
1. 参加者が写真をLINE Bot or Google Formにアップロード
2. 運営 or Bot が判定（必須2要素を確認）
3. 合格 → 次のヒント（YONでの曼荼羅完成場所）が届く
4. 不合格 → リトライ案内（「丸の内口のドームが写っていません」「八重洲側のビル群をもう一度」）

**物理方式（代替）:**
1. 参加者がYONで写真を見せる
2. スタッフが目視確認 → パスワードorスタンプ授与

### 撮影推奨位置（3候補）

1. **丸の内中央口前から八重洲方面を望む**
   - 駅舎のアーチ越しに八重洲の高層ビル群
   - 最もクラシックな構図

2. **日本橋口（北側）の通路から**
   - 丸の内側の赤レンガと新幹線高架が同時に見える
   - 線路含むので鉄道感強い

3. **東京駅地下1階グランスタから丸の内地下口を見上げる**
   - 上下の対比が面白いが、地下からの撮影になるので難易度高

### 参考: Var.25「黒真珠」の音楽的特徴

- ト短調（最初の短調変奏から3つ目 / 最も深い悲しみ）
- プトレマイオス天体説では「土星」= 時間の終わり
- 表向きは静かなアダージョ、内側に激しい情熱を秘める
- Goldbergの語源: Gold(金) + Berg(山/虫) = コガネムシ
- 対比: 金色(=金の玉/豊展観守像) vs 黒真珠(=悲しみ/終わり)

### 所要時間
- 東京駅到着〜撮影: 15-30分（駅構内の移動含む）
- 提出〜判定: 即時（Bot）or 5分（スタッフ確認）
