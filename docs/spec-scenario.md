# Stray Dog — 仕様書・シナリオ進行 v2

> **プロジェクト**: 神保町エリアの位置情報アドベンチャーゲーム
> **対象**: モバイルブラウザ（PWA対応）
> **スタック**: TypeScript + Vite 6 + Leaflet + Vitest
> **Deploy**: https://yon-straydog.vercel.app

---

## 1. 全体フロー

```
起動
  │
  ├─ [INTRO] タイピングアニメーション
  │   背景: 夫妻の写真(0.jpg) → 写真を差し出すシーンで犬の写真(gdog.png)に切替
  │   スキップボタン（画面下中央）
  │   終了 → 2×2パズルへ
  │
  ├─ [2×2 フォトパズル] 4枚のタイルを入れ替えて犬の写真を復元
  │   クリア → 「さぼうる の レインボウを さがせ」→ HUBへ
  │
  ├─ [HUB] 4スポットカード（2×2グリッド）
  │   未完了⚪ / 完了🟡 表示
  │   ストーリーモードボタン（📖）
  │
  ├─ [MAP] Leaflet地図 + GPS (モックモード自動フォールバック)
  │   スポットに接近 → 到着 → 「謎を解く」→ ミニゲーム起動
  │
  ├─ [MINI-GAME] スポットごとの謎解き
  │   クリア → バッジ獲得 → HUB
  │
  └─ [COMPLETE] 全3バッジ → s4（YON 3F）解放 → エンディング + 紙吹雪 + シェア
```

## 3. 4スポットとミニゲーム（v2）

| ID | スポット名 | アイコン | ミニゲーム | 内容 | 鍵条件 |
|----|-----------|---------|-----------|------|--------|
| s0 | YON 2F | 📖 | 2×2パズル | 犬の写真を復元（gdog-square.png 700×700） | 常に解放 |
| s1 | さぼうる | 🍨 | テキストクイズ | 「藍色のシロップの味は？」→ カルピス | 常に解放 |
| s2 | 響（野外彫刻） | 🔔 | 年代並び替え | バッハ4曲を古い順に並べる | s0 + s1 クリア |
| s3 | 神田橋公園 | 🗽 | 古層マッチング | 4人の偉人を earthdiver 4層に対応 | s0+s1+s2 クリア |
| s4 | YON 3F | 🎵 | なし（最終） | 到着で即ストーリー | 3バッジ |

### 3.1 2×2 パズル (`src/game/puzzle-starter.ts`)

- 犬の写真（gdog-square.png 中央クロップ 700×700）を4分割
- 操作: タップで選択 → 別タイルタップで入れ替え
- クリア → 「さぼうるのレインボウをさがせ」→ HUB

### 3.2 テキストクイズ (`src/game/text-quiz.ts`)

- 共通テキスト入力クイズエンジン
- `#quiz4-game` DOM を再利用
- 正解/不正解フィードバック + 1.2秒後自動クローズ

### 3.3 年代並び替え (`src/game/sort-game.ts`)

- 4アイテムをタップで選択 → 別アイテムタップで入れ替え
- 「確認」ボタンで判定
- `#simon-game` DOM を動的に再構築
- 響: 無伴奏チェロ組曲 → ブランデンブルク協奏曲 → マタイ受難曲 → ゴルトベルク変奏曲

### 3.4 古層マッチング (`src/game/match-game.ts`)

- earthdiver 4枚の絵画を古層バーとして表示（モナ・リザ/牛乳を注ぐ女/考える人/接吻）
- 3シリーズからランダム出題: **画家 / 文人 / 武将**
- `#puyo-game` DOM を動的に再構築
- 複数正解対応（altEras）
