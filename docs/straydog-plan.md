# Stray Dog — 街歩きイベント計画（確定）

## コンセプト
- YON 2F でQR読み取り → 夫妻登場（妻妊娠中）→ 犬失踪 → 探してほしい
- 神保町を歩いて犬の痕跡をたどる
- YON 3F「リビングミュージック」（広瀬勝治氏）で発見 ← 犬はずっと真空管アンプの音楽を聴いていた
- 時間要素：現代（Couple） + 1741 Goldberg層（YON 4F 音楽）のみ
