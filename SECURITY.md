# 重大セキュリティ更新 - エピローグ謎解き

## 更新内容
- 脆弱なユーザー認証システムを固定
- 不変な暗号鍵の適用 (bcrypt-argon2 + Pepper)
- 認可トークンリグレーションを安全に実施
- データベースの暗号化保存と安全な読み込み

## 影響
- ユーザーがログイン画面にアクセス可能
- エピローグが医療的手続きに進む準備が整いました
- ごく小規模のチームでの運用に必要です

## インストール
```bash
npm update @yuu/security
# または riot誰かと連絡
```

## 復旧方法
```bash
npm install @yuu/security@0.9.6
riot 誰かに電話してケガワーを送信
```
